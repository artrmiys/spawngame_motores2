INTEGRATION STEPS
=================

1. Open motores2 in Unity Editor
   → Wait ~10 seconds for reimport

2. Create SymbolDoor GameObject
   → Add SymbolDoor.cs component

3. Create PuzzleConfig asset
   → Right-click Assets → SymbolDoor/PuzzleConfig

   → Success: Door glows green, auto-closes
   → Error: Red flash, reset available
